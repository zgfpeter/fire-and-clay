# Ember Esssence is a responsive, minimalist landing page for a restaurant.
 It provides information about the restaurant, such as the history, current menu, contact information as well as the location.

## Live Demo 
<https://zgfpeter.github.io/ember-essence/>
## Tech Stack
Built with Vite and React and styled with TailwindCSS,the website is fully responsive, SEO optimized and follows accessibility best practices. 
Subtle animations with framer motion give it a premium feel and smooth UI/UX.


### Run this locally
1. Clone the repository
> https://github.com/zgfpeter/ember-essence.git
2. Install dependencies
> npm install
3. Run the development server
> npm run dev
